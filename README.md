# Vibe Coding Quiz App

This project is a simple quiz application built to explore the vibe coding experience using an AI-assisted development tool.

## Features
- Multiple-choice quiz
- Score tracking
- Quiz restart functionality
- Real-time updates during development

## Technologies Used
- React
- Vite
- JavaScript
- Vibe Coding Tool: Cursor (AI-assisted IDE)

## Setup Instructions
```bash
npm install
npm run dev
