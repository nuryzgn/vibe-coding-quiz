# Part 1 — Vibe Coding Tools (Research)

Below is a concise research list of modern “vibe coding” (agentic / conversational, context-aware) development tools.

## Cursor (Anysphere)
- **What it is:** AI-first code editor (VS Code–like) with chat + inline edits.
- **Key features:** project-wide context, chat/commands, multi-file edits, refactors, codebase Q&A.
- **Pricing:** free tier + paid plans (subscription).
- **Languages:** any language supported by VS Code ecosystems (JS/TS, Python, Java, C#, etc.).

## Windsurf (Codeium)
- **Company:** Codeium
- **Key features:** agentic “IDE” workflow, context-aware chat, multi-file changes, codebase understanding.
- **Pricing:** free tier + paid plans (subscription).
- **Languages:** broad IDE coverage (commonly JS/TS, Python, Java, etc.).

## Replit Agent (Replit)
- **What it is:** AI pair programmer inside Replit workspace.
- **Key features:** generates code, runs project, fixes errors, iterates with you.
- **Pricing:** free + paid tiers (depending on features/usage).
- **Languages:** many (JS/TS, Python, etc.) via Replit templates.

## v0.dev (Vercel)
- **What it is:** AI UI generator focused on React + web UI.
- **Key features:** generates UI components/pages from prompts, fast iteration for frontend.
- **Pricing:** free + paid tiers.
- **Languages:** primarily web UI (React/Next.js ecosystem).

## Bolt.new (StackBlitz)
- **Company:** StackBlitz
- **What it is:** AI full-stack app builder in the browser.
- **Key features:** generates/runs projects, edits multiple files, can scaffold full apps quickly.
- **Pricing:** free + paid tiers.
- **Languages:** web-focused stacks (JS/TS, Node, etc.).

## GitHub Copilot Workspace (GitHub)
- **What it is:** task-oriented Copilot workflow for planning + implementing changes.
- **Key features:** issue-to-plan, multi-file suggestions, repo context.
- **Pricing:** tied to Copilot plans (varies).
- **Languages:** broad GitHub ecosystem.

## Lovable (Lovable.dev)
- **What it is:** prompt-to-app builder aiming at rapid product prototypes.
- **Key features:** quick scaffolding, UI generation, iteration with chat.
- **Pricing:** free + paid tiers.
- **Languages:** typically web app stacks (varies by platform).
